The game code for "The Last Faun" for our group project for 2012->2013.
The project is for Unity3D v4.0<4.2

The project is a small demo of a game in which you play as a child who mets a Faun in the forest. Together the two characters must navigate through the puzzles using abilities only available to specific characters.

The first release of the game can be found at http://www.indiedb.com/games/the-last-faun
